### 🎓 [InhaCapstoneProject](https://github.com/InhaCapstoneProject) — RGB팀 · 2025 인하대학교 캡스톤 디자인

> **Flutter 앱 · Spring 백엔드 · Raspberry Pi · YOLOv4 객체 인식** 이 결합된 멀티 컴포넌트 시스템을 팀 프로젝트로 개발했습니다.
> 하드웨어(RaspberryPi)에서 수집한 영상을 컴퓨터 비전으로 분석하고, 서버를 통해 모바일 클라이언트로 전달하는 End-to-End 파이프라인을 설계·구현했습니다.
>
> *A multi-component capstone system combining a Flutter app, Spring backend, Raspberry Pi hardware, and YOLOv4-based object detection — an end-to-end pipeline from device to server to mobile client.*

**Key Features · 핵심 기능**
- 🥧 **엣지 디바이스 영상 수집** — Raspberry Pi 에서 카메라 영상을 캡처·전처리하여 업로드
- 👁 **YOLOv4 객체 인식** — Jupyter 기반 학습/추론 파이프라인으로 커스텀 객체 탐지 모델 운용
- ☕ **Spring 중계 서버** — 엣지와 모바일 클라이언트 사이 데이터 흐름을 중계·관리
- 📱 **Flutter 모바일 앱** — Dart·Flutter 기반 크로스 플랫폼 UI 로 인식 결과 확인
- 🔌 **End-to-End 파이프라인 설계** — 센서(RasPi) → CV(YOLO) → 서버(Spring) → 클라이언트(Flutter) 흐름 통합

| Component | Stack | Repository |
|---|---|---|
| 📱 Mobile Frontend | Dart · Flutter | [frontend](https://github.com/InhaCapstoneProject/frontend) |
| ☕ Backend Server | Java · Spring | [backend](https://github.com/InhaCapstoneProject/backend) |
| 🥧 Edge Device | Python · Raspberry Pi | [RaspberryPI](https://github.com/InhaCapstoneProject/RaspberryPI) |

**Key Features · 핵심 기능**

- 🌐 **웹 프론트엔드** — 프롬프트 입력 UI 와 생성 결과 미리보기 제공 (JavaScript)

- 🔧 **TypeScript 백엔드** — 웹·Unity 양쪽 클라이언트 요청을 받아 NanoBanana3 생성 파이프라인을 중계
